# Inventory Project 

## Features
* Login/Logout
* Product
    * List with pagination
    * Store product
    * Delete Product
* Purchase Product


### Requirements :
_php- 8.1, mysql- 8, nodeJs- 20.0, npm- 9.6_

## Installation
***

* Clone the repository
* Change to the project directory
* connect database & run  

<pre><code> 
composer install
npm install
npm run dev
php artisan migrate
php artisan db:seed
 
</code></pre>
